## テスト分類別カバレッジ

- 正常系: 21項目（50%）
  - ページアクセス、データ表示、タスク完了、ボタンクリック、空データ
- 異常系: 7項目（17%）
  - エラー処理、ネットワークエラー、データ形式エラー
- セキュリティ: 5項目（12%）
  - 認証、セッション、XSS、CSRF
- レスポンシブ: 5項目（12%）
  - デスクトップ、タブレット、モバイル、タッチ操作、画面回転
- ワークフロー: 3項目（7%）
  - 連続操作、データリフレッシュ、状態保持
- エッジケース: 3項目（7%）
  - 長いテキスト、進捗率極値
